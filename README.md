# 🔋 NASA Battery EDA
Exploratory Data Analysis of the NASA PCoE Battery Dataset (B0005–B0018)
as part of a QUBO-based quantum optimization research series for battery SOH prediction.

## Results
| Battery | Discharge Cycles | Final SOH |
|---------|-----------------|-----------|
| B0005   | 168             | ~70%      |
| B0006   | 168             | ~65%      |
| B0007   | 168             | ~78%      |
| B0018   | 132             | ~70%      |
